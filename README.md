# CONVOLUTIONAL-AUTOENCODER-FOR-IMAGE-DENOISING

## Project Overview

This project implements Image Denoising using a Deep Learning model.
The model learns to remove noise from images and reconstruct a clean version of the image.

Image denoising is an important task in computer vision used in areas such as medical imaging, photography, and image restoration.

## Technologies Used

Python

PyTorch

NumPy

Matplotlib

Jupyter Notebook

## Project Files

image denoising.ipynb – Main notebook containing the implementation

Dataset – Noisy and clean images used for training

README.md – Project description

## Workflow

Load image dataset

Add noise to images

Train a deep learning model to remove noise

Test the model on noisy images

Compare noisy and denoised outputs

Result

The trained model reduces noise in images and produces a cleaner reconstructed image.
